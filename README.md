## вңҰ Features

- **Matrix Rain** вҖ” Canvas-based falling character animation in the Hero section
- **Typing Animation** вҖ” Cycles through roles with a blinking cursor effect
- **Glitch Text** вҖ” RGB-split CSS glitch effect on section headings
- **Animated Skill Bars** вҖ” Scroll-triggered progress bars with percentage labels
- **CountUp Stats** вҖ” Animated number counters in the About section
- **Scroll Progress Bar** вҖ” Neon line at the top tracking page scroll position
- **Active Section Nav** вҖ” Navbar highlights the current section using IntersectionObserver
- **Floating Social Dock** вҖ” GitHub / LinkedIn / WhatsApp icons pinned to the right side
- **Back to Top Button** вҖ” Appears after scrolling 400px with a cyberpunk clip-path style
- **Film Grain Overlay** вҖ” Subtle noise texture for that analog terminal feel
- **Scanline Effect** вҖ” Animated scanline sweep across the page
- **EmailJS Contact Form** вҖ” Functional contact form with real email delivery
- **Project Tag Filter** вҖ” Click tags to filter projects by technology
- **Fully Responsive** вҖ” Mobile-first layout with a collapsible nav menu

## вңҰ Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/mithradevtv-droid/creative-portfolio.git
cd creative-portfolio

# Install dependencies
npm install
```

### Environment Setup

Create a `.env` file in the root directory:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

> Get these from [EmailJS Dashboard](https://www.emailjs.com/) вҖ” it's free for up to 200 emails/month.

### Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder вҖ” ready to deploy anywhere.

## вңҰ Deployment

This project deploys easily to **Vercel** (recommended):

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or push to GitHub and connect the repo to [vercel.com](https://vercel.com) for automatic deployments on every push.

> вҡ пёҸ Remember to add your `VITE_EMAILJS_*` environment variables in the Vercel dashboard under **Settings вҶ’ Environment Variables**.

## вңҰ Customization

- **Your info** вҖ” Update name, bio, links, and project details in each component
- **Colors** вҖ” Edit CSS variables in `src/index.css` under `@theme`
- **Projects** вҖ” Edit the `projects` array in `src/components/Projects.tsx`
- **Skills** вҖ” Edit the `skillCategories` array in `src/components/Skills.tsx`
- **Stats** вҖ” Edit the `stats` array in `src/components/About.tsx`
